# Code of Conduct

We want this repo to be a serious but friendly place to build creative AI infrastructure.

## Expected Behavior

- Be direct, respectful, and specific.
- Critique outputs and code, not people.
- Share enough evidence for others to reproduce your claim.
- Credit contributors for ideas, examples, and proof artifacts.
- Keep generated media and cultural references thoughtful, especially in country/faction satire.

## Unacceptable Behavior

- Harassment, threats, hate speech, or personal attacks.
- Sharing secrets, private keys, private user data, or non-public production artifacts.
- Spam, low-effort issue flooding, or unrelated promotion.
- Deliberately submitting misleading proof or copied outputs without attribution.

## Reporting

Report concerns privately to the maintainers through GitHub or the project community channel once published.

Maintainers may edit, hide, close, or remove contributions that violate this code.
